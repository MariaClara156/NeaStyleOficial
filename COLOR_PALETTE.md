# Paleta de Cores NeaStyleOficial

## Cores Principais

| Cor | Código | Uso |
|-----|--------|-----|
| Preto Escuro | `#1c1c1c` | Fundo do rodapé, texto em fundos claros |
| Branco | `#ffffff` | Texto em fundo escuro (rodapé), destaques |
| Bege Quente | `#E8d8c3` | Botões primários, títulos do rodapé, destaques |
| Verde Cinzento | `#A3b18A` | Links padrão, elementos secundários, ícones |
| Cinza Claro | `#f2f2f2` | Fundo alternativo, ícones sociais |

## Harmonia e Aplicação

### Rodapé
- **Fundo**: `#1c1c1c` (preto escuro)
- **Texto principal**: `#A3b18A` (verde cinzento)
- **Títulos (h6)**: `#E8d8c3` (bege quente)
- **Links**: `#A3b18A` → hover `#E8d8c3`

### Elementos Interativos
- **Links padrão**: `#A3b18A`
- **Botões primários**: Fundo `#E8d8c3`, texto `#1c1c1c`
- **Navegação ativa**: Fundo `#A3b18A`, texto `#ffffff`
- **Bordas**: `#E8d8c3`

### Ícones Sociais
- **Fundo padrão**: `#f2f2f2`
- **Ícones**: `#A3b18A`
- **Hover geral**: Fundo `#E8d8c3`, texto `#1c1c1c`
- **Hover específicas**: Todas as redes usam `#A3b18A` (exceto dribbble com `#E8d8c3`)

## Princípios de Design

✓ **Contraste**: Cores escuras vs claras garantem legibilidade  
✓ **Harmonia**: Tons terrosos e neutros criam ambiente acolhedor  
✓ **Acessibilidade**: Verde cinzento e bege oferecem bom contraste  
✓ **Modernidade**: Paleta minimalista e sofisticada
