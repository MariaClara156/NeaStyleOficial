## Overview

This project is an e-commerce platform developed as part of the Systems Analysis and Development degree program.

The system aims to provide a complete online shopping experience for customers while allowing administrators to manage products, inventory, and sales through a dedicated administrative area.

The project was developed following Software Engineering principles, including Requirements Engineering, Database Modeling, Software Architecture, UI/UX Design, and Agile Methodologies.

---

## Objectives

- Sell clothing products for teenagers
- Manage inventory and product catalog
- Support customer purchases
- Provide administrative management tools
- Apply concepts learned throughout the degree program

## Architecture

The application follows the MVC (Model-View-Controller) architectural pattern.

- **Models** represent business entities.
- **Views** handle the user interface.
- **Controllers** process requests and business logic.

---

## Database

The database was designed using Entity-Relationship Modeling.

Main entities:

- User
- Customer
- Administrator
- Product
- ShoppingCart
- CartItem
- Favorites
- Order
